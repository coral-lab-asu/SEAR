# No Universal Prompt

Project website for "No Universal Prompt: Unifying Reasoning through Adaptive Prompting for Temporal Table Reasoning"

ArXiv: https://arxiv.org/abs/2506.11246

## Website Structure

- index.html - Main landing page
- static/ - CSS and JavaScript files
- assets/ - Images, tables, and figures (placeholders for now)

## Local Development

Open index.html in a web browser to view the site locally.

## Deployment

This site is designed to be hosted on GitHub Pages.
